Simetria Studio 3D  

**Projeto Acadêmico – Tech Academy 2 (2025/4 EAD)**  
**Curso:** Tecnologia em Análise e Desenvolvimento de Sistemas (TADS)  
**Instituição:** Faculdade Integrado de Campo Mourão – PR  

Descrição do Projeto

Site institucional desenvolvido para **Simetria Studio 3D**, estúdio especializado em **arquivos STL exclusivos para joalheria 3D**.  
O objetivo é automatizar a vitrine de produtos, agilizar orçamentos e profissionalizar a presença digital do cliente, reduzindo a dependência de redes sociais (Instagram/WhatsApp).

Repositório: https://paulozaka.github.io/simetria/

Link no InfinityFree: https://simetria-3d.xo.je

Tecnologias Utilizadas

| Camada | Tecnologia |

| **Back-end** | PHP 8.2 (single-entry point) |
| **Front-end** | HTML5, Bootstrap 5.3, SCSS, AOS Animations |
| **Rotas** | `.htaccess` + `$_GET['param']` + `explode()` |
| **Dados** | Array em `dados.php` (persistência em memória) |
| **Validação** | JavaScript nativo (HTML5 + `checkValidity()`) |
| **Versionamento** | Git + GitHub (commits graduais) |

Funcionalidades Implementadas

- URLs amigáveis (`/lancamentos`, `/produto/3`)  
- Carrossel responsivo com GIFs animados (home)  
- Produtos dinâmicos com destaque e detalhes  
- Formulário de contato com 6 campos + validação JS completa  
- Página de erro 404 personalizada e responsiva  
- Totalmente responsivo (mobile-first)  
- Estrutura de pastas organizada e limpa
